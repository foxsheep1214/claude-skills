## docx 读取方法

⚠️ **必须用 terminal() 调用 venv python，不能用 execute_code**：

```bash
~/.hermes/skills/docx/venv/bin/python -c "
from docx import Document
doc = Document('/path/to/file.docx')
for para in doc.paragraphs:
    if para.text.strip():
        print(para.text)
for i, table in enumerate(doc.tables):
    for row in table.rows:
        print(' | '.join([cell.text.strip() for cell in row.cells]))
"
```

## 章节编号重排规则

当从参考文档迁移结构到 Tracker II 需求论证时：
1. 先读取参考文档提取结构
2. 对比 Tracker II 现有章节找出缺口
3. 用 patch 按标题匹配迁移（不用编号匹配，编号在重排中会变化）
4. 重排后执行验证四查（见 SKILL.md 主文件）
