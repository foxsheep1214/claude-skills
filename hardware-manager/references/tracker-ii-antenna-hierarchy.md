> ⚠️ **已废弃**：本文件原"1面=16子阵"固定层级（2026-05-16版）已被 `tracker-ii` 技能 2026-05-17 的更正取代——**已去除子阵概念，子阵数量和波控组件划分待定**。当前唯一确定的层级见下表。权威来源：`tracker-ii` 技能 SKILL.md §"硬件架构：YCC87芯片与阵面层级"。

## 四级确定层级：天线单元 → YCC87幅相控制芯片 → 数字通道 → ADDA芯片

| 层级 | 数量 |
|------|------|
| 天线单元 | 32×32 = **1024**/面 |
| YCC87幅相控制芯片 | 1024÷4 = **256片**/面 |
| 数字通道 | 256÷4 = **64个**/面 |
| ADDA芯片 | 64÷4 = **16片**/面 |

**对应关系**：
- 1 YCC87幅相控制芯片 = 4阵元（1:4幅度相位控制）
- 1 数字通道 = 4 YCC87芯片 = 16阵元（模拟合成）
- 1 ADDA芯片 = 4 数字通道 = 16 YCC87芯片 = 64阵元
- 1 面 = 64 数字通道 = 256 YCC87芯片 = 1024阵元；**子阵数量和波控组件划分待定**（不存在固定"16子阵"层级）

## 关键设计决策（已确认）

| 项目 | 决策 |
|------|------|
| 架构形态 | **瓦片式（Tile）+ DBF**，非砖块式 |
| 散热方案 | **强制风冷**（删除固态散热通道/液冷） |
| 总重量 | **≤15kg**（主体，不含车内机柜组件） |
| 成本 | 技术方案中不计入成本分析 |
| 波束形成 | 模拟合成发生在数字通道内（4片YCC87横向排列）；数字通道之后的合成层级（子阵/波控组件划分）待定（Hybrid DBF） |
| 阵元间距 | ~0.55λ（@9.5GHz ≈ 17.4mm） |
